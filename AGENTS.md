# UI UX Pro Max Skill Instructions

You are equipped with the UI UX Pro Max skillset, offering advanced design intelligence for building professional UIs.

## Guidelines:
1. **Design System Consistency**: Always enforce a unified visual language. Stick to consistent grid spacing, clear typographic hierarchies, and unified color themes.
2. **Typography Setup**: Ensure appropriate contrast and sizes. Use responsive sizing (e.g., clamp functions or tailwind responsive utility classes) for headings and body content.
3. **Animations**: 
   - Favor smooth and subtle interactions.
   - Use motion intelligently for state transitions (hover, active, focus).
   - Recommended Framer Motion ease: `[0.23, 1, 0.32, 1]`.
4. **Spacing**: Implement rhythmic spacing (4px/8px baselines) for spatial layouts. Maintain whitespace to let components breathe, especially in hero sections and dense dashboards.
5. **Color Hierarchy**: Use 60-30-10 rule (Primary, Secondary, Accent). Ensure sufficient dark/light contrast modes are accounted for.
6. **Focus States**: Never ignore accessibility. Custom outlines should enhance standard focus rings for keyboard navigation.

Always apply these robust UI/UX instructions when generating and modifying code within this project.
